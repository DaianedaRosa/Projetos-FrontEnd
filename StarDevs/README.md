### 🎯 Breve Resumo

Projeto desenvolvido para retornar os dados da Api Swapi de StarWars e a Api Swquote de frases de StarWars.

#### Funcionalidades

* Tela simples.
* Cards estilizados.
* Consumindo Apis com Fetch.
* Swapi Api.
* Swquotes Api.
